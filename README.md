# Praktikum Exception Handling

Kumpulan program untuk Percobaan 1-12 sesuai modul "10. Praktikum Exception Handling".

## Cara menjalankan

Setiap file bisa di-compile dan dijalankan sendiri-sendiri, contoh:

```
javac Percobaan1_Sebelum.java
java Percobaan1_Sebelum

javac Percobaan1_Sesudah.java
java Percobaan1_Sesudah
```

Lakukan hal yang sama untuk file lainnya. Urutan yang disarankan untuk
mengerjakan dan screenshot:

| Percobaan | File | Catatan |
|---|---|---|
| 1 | Percobaan1_Sebelum.java, Percobaan1_Sesudah.java | Nama class diganti dari "Exception" karena bentrok dengan java.lang.Exception (lihat komentar di file) |
| 2 | Percobaan2_Sebelum.java, Percobaan2_Sesudah.java | Versi "Sesudah" bisa infinite loop, tekan Ctrl+C untuk berhenti (lihat komentar di file) |
| 3 | Percobaan3_Sebelum.java, Percobaan3_Sesudah.java, Percobaan3_Sesudah_MultiCatch.java | Ada 2 versi perbaikan: catch umum & multi-catch |
| 4 | Percobaan4_Sebelum.java, Percobaan4_Sesudah.java | Urutan statement di try ditukar |
| 5 | Percobaan5.java | Tugasnya cuma jalankan + analisa (analisa tertulis ada di komentar file) |
| 6 | Percobaan6.java | Tugasnya cuma jalankan + analisa; ada baris unreachable code di soal asli, sudah dijelaskan di komentar |
| 7 | Percobaan7.java | Tugasnya cuma jalankan + analisa |
| 8 | Percobaan8_Sebelum.java, Percobaan8_Sesudah.java | Program awal crash (uncaught), versi perbaikan pakai try-catch-finally |
| 9 | Percobaan9_Sebelum.java, Percobaan9_Sesudah.java | Versi sesudah pakai string kosong "" sesuai instruksi soal |
| 10 | Percobaan10.java | Akan membuat file books.txt otomatis di folder yang sama |
| 11 | Percobaan11.java | Custom exception extends Throwable |
| 12 | Percobaan12.java | Custom exception extends Exception |

## Ringkasan analisa untuk Percobaan 5, 6, 7 (yang diminta "berikan analisa")

Analisa lengkap ada langsung di komentar bagian atas tiap file
(Percobaan5.java, Percobaan6.java, Percobaan7.java), supaya gampang
di-copy kalau perlu dilampirkan sebagai jawaban tertulis juga.
